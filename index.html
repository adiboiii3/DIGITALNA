🔌 Multivibratori — Vizuelno

Interaktivno objašnjenje astabilnog i monostabilnog multivibratora s animiranim shemama i signalima

// Šta je multivibrator?

Multivibrator je elektronski sklop koji koristi povratnu spregu između dva tranzistora da bi prelazio između dva stanja. Svaki tranzistor radi kao elektronski prekidač — ili je potpuno uključen (zasićenje) ili potpuno isključen (zakočenje).

  • Astabil 0 stabilnih stanja — sam beskrajno osciluje
  • Monostabil 1 stabilno + 1 kvazistabilno stanje
  • Bistabil 2 stabilna stanja — flip-flop, treba 2 okidača

// Shema astabilnog multivibratora
// Komponente i uloge
  • RC1 RC2 Kolektorski otpornici — ograničavaju kolektorsku struju, red veličine kΩ
  • RB1 RB2 Bazni otpornici — kroz njih se prazne kondenzatori, red veličine 10·kΩ → određuju trajanje kvazistabilnih stanja!
  • C1 C2 Kondenzatori — propuštaju brze promjene → pozitivna sprega → srce osciliranja
  • TR1 TR2 NPN bipolni tranzistori — rade kao elektronski prekidači (zasićenje/zakočenje)
// Zašto je sprega POZITIVNA?

Kondenzator propušta samo brze promjene napona. Kad napon na kolektoru TR1 naglo padne — ta promjena se ODMAH prenosi na bazu TR2.

TR1 počne voditi više
→ Kolektor TR1 pada
→ C2 prenosi pad na bazu TR2
→ TR2 se isključuje
→ Kolektor TR2 raste
→ C1 prenosi rast na bazu TR1
→ TR1 se još više uključuje → LAVINA!
// Dvije brzine promjene

  • RB·C Spora promjena — kondenzator se PRAZNI kroz veliki bazni otpornik (10·kΩ) → određuje trajanje stanja

// ⚡ Ključna razlika vs Astabil
  • Astabil Samo pozitivna sprega u OBA smjera → nema stabilnog stanja
  • Monostabil Pozitivna u jednom (CB) + negativna u drugom (R1-R2) → jedno stabilno!
// Zašto RB vezan na UCC?

  • IBzas1 UCC → RB → Baza TR1 → Masa

// Struje u kvazistabilnom stanju
  • IBzas2 UCC → RC1 → R1 → R2 → Baza TR2
    TR2 zasičen, TR1 zakočen
  • Iprazn UCC → RB → CB (prazni) → TR2 → Masa
    τ = RB · CB → određuje trajanje T!
T = 0.693 · RB · CB
// Zašto UCE1 raste BRZO, a UBE1 raste SPORO?

UCE1 — skoro trenutni skok

U kolektorskom kolu TR1 nema kondenzatora — samo otpornici (RC1, R1, R2). Bez kondenzatora nema RC vremenske konstante → nema eksponencijalnog oblika → brz skok!

UBE1 — sporo eksponencijalno

CB se prazni kroz RB. RB je velik (10·kΩ) → mala struja → sporo pražnjenje → eksponencijalni rast napona na bazi TR1. Kad UBE1 dostigne Uγ → TR1 provodi → povratak u stabilno!

UBE2 — nema eksponencijale!
U kolu koje određuje potencijal baze TR2 (djelitelj R1-R2) nema kondenzatora → samo otpornici → nema RC konstante → promjena napona je skoro trenutna.

// Stacionarni registar — paralelni upis, paralelni ispis

Svaki D flip-flop je neovisan — ima vlastiti ulaz i izlaz. Nema fizičke veze između flip-flopova osim zajedničkog CLK signala. Na uzlaznu ivicu CLK-a svaki flip-flop "uhvati" vrijednost sa svog ulaza D.

// Interaktivni vremenski dijagram

Klikni na ivice CLK-a ispod da vidiš kako svaki FF uzima svoju vrijednost nezavisno

Pritisni Sljedeća ivica da vidiš šta se dešava na svakom taktu.
// Ključna pravila
  • D ff Koristimo D flip-flop jer direktno prenosi ulaz na izlaz — Q = D
  • CLK↑ Promjena se dešava SAMO na uzlaznoj ivici CLK-a
  • Neovisno Svaki FF rješavamo zasebno — dijagram po dijagram
  • Ignorisanje Promjena D između dvije ivice — FF je ne vidi!
// Pomjerački (Shift) registar — serijski upis

Izlaz svakog FF je ulaz sljedećeg. Podatak putuje kroz registar jedan korak po taktu. Za n FF-ova, podatak sa ulaza pojavi se na izlazu nakon n taktova.

// Zašto koristimo PRETHODNO stanje?
U trenutku uzlazne ivice CLK-a — signal sa izlaza prethodnog FF-a još nije stigao da se prenese na novi izlaz.
→ Svaki FF "gleda" stanje koje je prethodni FF IMAO PRIJE te ivice.
→ Zbog toga rješavamo ivicu po ivicu, a ne FF po FF!

// Prstenasti (Ring) brojač

Izlaz Q posljednjeg FF-a vraća se na ulaz prvog. Sa n FF-ova — n stanja, dijeli frekvenciju sa n.

// Johnsonov brojač

Jedina razlika: vraća se Q̄ (komplement) posljednjeg FF-a. Sa n FF-ova — 2n stanja, dijeli frekvenciju sa 2n.

Ring: Q → ulaz FF0 → n stanja
Johnson: Q̄ → ulaz FF0 → 2n stanja
Zašto 2n? Komplement "hvata" i fazu punjenja i fazu pražnjenja.
// Interaktivni simulator — Ring vs Johnson
Ring: 4 FF-a → 4 stanja. Johnson: 4 FF-a → 8 stanja.
// Bidirekcioni registar — pomak lijevo / desno

Ulaz DIR određuje smjer pomaka. DIR=1: ulaz sa lijeve strane (SIEL), pomak desno. DIR=0: ulaz sa desne strane (SIER), pomak lijevo.

// Asinhroni binarni brojač (JK flip-flop, silazna ivica)

FF0 reaguje na svaku silaznu ivicu CLK-a. Svaki naredni FF reaguje samo na silazni pad Q prethodnog. Npr. FF1 se mijenja samo kad Q0 ide s 1→0.

// Brojač nazad — jedina razlika
Naprijed: Q prethodnog → CLK sljedećeg
Nazad: Q̄ (komplement) prethodnog → CLK sljedećeg
→ Kad Q ide 0→1, Q̄ ide 1→0 (silazna!) → okida sljedeći odmah
// Interaktivni 4-bitni binarni brojač
Pritisni Takt impulsa ili Auto da pokreneš brojač.
// Dekadni brojač (Modul 10) — NI kolo kao "kočnica"

Kada se pojavi binarna kombinacija 1010 (=10), NI kolo daje nulu na izlazu i momentalno resetuje sve FF-ove nazad na 0000. Kombinacija 10 je vidljiva samo na kratko (ns).
